Yeoman and Node.js

Yeoman builds the framework for projects. You have to use the terminal to install Yeoman as well as to build your projects through Yeoman. Yeoman consists of three types of tools: yo, build tool(Grunt), and the package manager(npm)

npm has to be installed separatly from Yeoman. npm is the package manager for Node and comes bundled with it. To use a new generator, you have to install it from npm first.

Yo will present the option for Bootstrap, Sass, and Modernizr without extra installation. Most generators you will use in the beginning do not require extra installation, so Yoeman should be the first installation.

Compass and Ruby

Compass uses Sass. Sass extends CSS adding nested rules, variable, selector inheritance. etc.



Ruby comes with compass, you have to install Compass.



Coding in Sass makes coding simpler and easier to read. I recommend glossing over Sass first. Its not complicated but has some cool features.

Ruby Compass

Assemble, HandleBars, and Bootstrap

Assemble is a static site generator for Grunt.js and Yeoman.



Handlebars is a semantic web template system. Handlebars works by taking HTML and Handlebar expressions and compiles them to a JavaScript function.



Bootstrap is a HTML, CSS and JS framework. Useful for front end development.
